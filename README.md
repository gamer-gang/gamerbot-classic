## Development

### Setup

- install docker and docker-compose
- copy `.env.example` to `.env` and fill in all variables
- install dependencies: `yarn`
- start: `yarn dev`

### Debugging (VS Code)

- start as usual: `yarn dev`
- attach to process using `dev` configuration
